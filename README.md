# Coreto Staking

## Development

### 1. Initialize pnpm ( skip if pnpm is already installed )

```sh
npm run env-init
```

### 2. Run development server

```sh
pnpm dev
```

## Deployment

### 1. Invoke production build

```sh
pnpm build
```

### 2. Run local preview for QA ( optional )

```sh
pnpm preview
```
